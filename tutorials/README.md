# tutorials

Source files accompanying tutorials published on the [Ars Mathematica website](https://ro-smtz.github.io/ars-mathematica/). Each subfolder matches a tutorial page and contains the final `.tex` file discussed in the tutorial.

## Structure

```text
tutorials/
├── YYYY-tutorial-title/
│   ├── main.tex
│   └── README.md   ← link to the corresponding website tutorial
└── ...
```
